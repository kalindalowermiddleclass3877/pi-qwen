## 🚀 Getting Started

Once you install the extension, you must update your current session to include these new features. Open your pi interface on your computer.

1. Type `/reload` in your chat window and press Enter. This step refreshes the program state.
2. Type `/login qwen` in the command window.
3. Your web browser will open automatically.
4. Sign in with your Qwen credentials.
5. Confirm the request in your browser if a prompt appears.

## ⚙️ Selecting Your Model

The extension gives you access to a specific coding model. You must select this model after you log in.

1. Type `/model` into your chat interface.
2. A list of choices appears on your screen.
3. Find and click `qwen/coder-model`.
4. Your setup is now complete. You can begin typing your coding questions. The assistant will now use the Qwen logic to generate responses.

## 💡 Using the Extension

You do not need to configure complex network settings. The software handles the technical details for you. It automatically prepares your messages to match the format that Qwen expects. This prevents errors when you ask for code segments or technical explanations.

If you face connection issues, check that your internet firewall allows the application to reach the internet. You can repeat the `/login qwen` command at any time if your session expires.

## 📁 Technical Details

This extension simplifies the interaction with remote servers. 

- The software connects to the official Qwen authentication gateway.
- It identifies your account permissions through the secure device-code flow.
- It translates your chat requests into a format compatible with the provider.
- It only supports the `coder-model` path.

This design keeps the workspace clean. It removes the need for manual API keys or complex environment variables inside your system settings. The tool acts as a silent helper behind the scenes of your coding project.

## 🛡 Security Practices

This application follows standard security protocols. It does not store your password on your local hard drive. Instead, it uses a temporary token system to remember your login status. If you suspect any unauthorized access, revoke the application permission through your Qwen account settings page. Always download the software from the official link provided in this document to ensure the integrity of your installation.

## 🔧 Troubleshooting

If the commands do not work as expected, verify the following:

- Ensure the pi application version is up to date.
- Restart the application if the `/model` command returns an error.
- Check that your system clock matches the current time. Time mismatches cause issues with secure server handshakes.
- Close other applications that might block network traffic if you experience time-outs.

If you encounter persistent issues, perform a clean reinstall. Delete the old files, download a fresh copy from the release link, and run the installer again. This process resets your local configuration files to their default state.

## 📋 Compatibility

This version works on current editions of Windows. It requires the base pi package to function. The extension does not modify your core computer settings. It stays contained within the scope of the pi extension folder. You can remove the extension at any time by deleting the relevant folder within the pi installation directory.
